Directory Contents

1) aws - contains all logs and output for both the configuration run on aws

2) source contains build.sbt, makefile, PageRank.scala, WikiParser.java 

3) Report.pdf

4) readme.txt

TO RUN THE PROGRAM
Update the make file first. Update the input and output paths from the following line in the file 
i.e sbt "run <input_path> <output_path>" 

To run locally from the Terminal go to the source directory and run "make" command.
